## 脚本使用方法

需要安装 `Python 3.7` 及以上版本。然后，直接在控制台调用脚本即可：

```shell
python Markdown转为xlsx.py
```

即可将 `.md` 文件转为 `.xlsx` 表格，并在控制台展示当前的情况（形如上面展示的列表）。

## 脚本使用方法

歌曲调号首先会被记录在 `歌曲调号总结.md` 中。如果需要进行分析及可视化，可以先将其转为 `.db` 文件：

然后，可以运行 `visualize.py` 进行数据的进一步分析，包括生成表格以及扇形图。

```shell
python md2sql.py
python visualize.py
```
